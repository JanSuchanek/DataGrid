# ndatagrid

DataGrid component — sorting, filtering, inline editing, mass actions, pagination.

## Key Classes
- `DataGridControl` + `DataGridControlFactoryInterface`
- DI Extension: `DataGridExtension`
- Extends Contributte/Datagrid

## Conventions
- PHP 8.2+ with `declare(strict_types=1)`
- PSR-4 autoloading in `src/`
- Nette DI Extension for service registration
- PHPStan level 9
- Part of `jansuchanek/*` on Packagist
- GitHub: https://github.com/JanSuchanek/ndatagrid
